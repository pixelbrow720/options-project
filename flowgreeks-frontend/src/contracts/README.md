# src/contracts — backend type mirror

This folder is **read-only** from the frontend's perspective. The
canonical source is `flowgreeks-engine/contracts/`. Files under
`./types/`, `./samples/`, and `openapi.json` are overwritten by
`pnpm sync:contracts` (see `scripts/sync-contracts.ts`).

If you need a frontend-only derived type (e.g. UI-state union, narrowed
view-model), put it in `src/shared/` or the feature folder — never
edit anything in here.

## Sync workflow

When the backend payload shape changes:

1. Edit `flowgreeks-engine/contracts/types/snapshot.ts` in the engine
   repo and run `bash scripts/export_contracts.sh` there.
2. In this repo: `pnpm sync:contracts`.
3. `pnpm typecheck` — every type drift surfaces as a compile error.

## Layout

- `types/snapshot.ts` — full TS contract for REST envelopes + WS frames.
- `samples/` — fixture JSON pulled from real Databento data (used by
  Storybook and test mocks when the backend isn't running).
- `openapi.json` — generated by FastAPI; useful for openapi-typescript
  if we ever switch to generated client code.
